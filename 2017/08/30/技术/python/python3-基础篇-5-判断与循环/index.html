while循环

当满足条件时执行循环体，while循环执行完后再调回到 while 所在的位置，如此重复进行，直到 while 表达式为 False 为止，注意是while所在的位置
使用while循环的三个要求：

  1. 尽量少用 while-loop ，大部分时候 for-loop 是更好的选择。
  2. 重复检查你的 while 语句，确定你测试的布尔表达式最终会变成 False 。
  3. 如果不确定，就在 while-loop 的结尾打印出你要测试的值。看看它的变化。

注意：如果while循环不能像for循环那样，在集合被穷尽之后停止下来，我们又怎么样才能控制while循环呢？
方式一：在循环过程中制造某种可以使循环停止下来的条件，增加break、continue之类的命令。
方式二：改变使循环成立的条件。比如直接指定False或产生False效果的条件

  • 跳出结束循环：break （只用于循环）
  • 重新开始循环：continue （只用于循环）
