enumerate() 枚举出索引值与对应的元素
循环列表时获取元素的索引（通过enumerate函数实现）
方式一：直接在索引号加1的方式实现（因为索引是从0开始计数，所以加1）
letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
for num,letter in enumerate(letters):
print(num + 1,letter)

方式二： 指定enumerate第二个参数实现
letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
for num,letter in enumerate(letters,1):
print(num,letter)

sorted() 排序，sorted函数按照长短，大小，英文字母的顺序给每个列表中的元素进行排序,sorted函数并不会改变列表本身，你可以把它理解成是先将列表进行复制，然后进行顺序的整理
正向排序（默认）
num_list=[6,2,7,4,1,3,5]
print(sorted(num_list))
结果：[1, 2, 3, 4, 5, 6, 7]

反向排序
num_list=[6,2,7,4,1,3,5]
print(sorted(num_list,reverse=True))
结果：[7, 6, 5, 4, 3, 2, 1]

类方法

字符串类型

str方法名 功能
str.capitalize() 首字母大写
str.count(‘a’) 统计a这个字符串的个数
str.center(50,’-‘) 一共打印50个字符，把字符放中间，不够的用-补上，方便写华丽的分隔符
str.encode() 把字符串转换成二进制
str.endswith(“ex”) 判断一个字符串是否以ex结尾，比如你想判断一个邮件地址是否以.com结尾
str.startswith(“ex”) 判断一个字符串是否以ex开头
str.expandtabs(tabsize=10) 如果你的字符串有写\t 这表示tab键盘，会根据你指定的值，比如这里写的是10而将tab扩展到10个空格
str.find() 查找所提供字符串的起始索引号
如:
str="mengma.com"
print(str.find("ng"))
>>>2
str.format() 格式化字符串，用于填空，其实就是代入变量的一种方式跟格式定位符%类似，需要注意的是需要代入的位置是以{}来定位（比如网址填空，其实使用另一种方法，即格式定位符也是可以实现的）
str.format_map() 跟format类似，它是用传字典进去，当然也要对应上值
str.index() 取索引值
str.isalnum() 判断是否是阿拉伯数字或字母（不包括特殊字符）
str.isalpha() 判断是否是英文字符
str.isdecimal() 判断是否是十进制
str.isdigit() 判断是否为数字，如果是数字则为True，不是数字则为False
str.isidentifier() 判断是否是一个合法的变量名
str.islower() 判断是否是小写
str.isnumeric() 判断是否是数字字符
str.isspace() 判断是否是空格
str.istitle() 判断是否是标题，必须全部字的首字母大写才是认为是标题
str.isprintable() 判断是否可打印
str.isupper() 判断是否是大写
str.join() 将列表中的内容合并成一个单一的字符串，比如’+’.join(‘1’,’2’,’3’)
str.ljust(50,’*’) 跟center类似，保证为50个字符，左边不动，右边不够的以*号补全
str.rjust(50,’-‘) 跟ljubs相反
str.lower() 变小写
str.upper() 变大写
str.strip() 默认去掉2边的空格或回车
str.lstrip() 去掉左边的空格或回车
str.rstrip() 去掉右边的空格或回车
str.maketrans(“abcde”,’123456’) 人为制造的索引，就像一个密码本，配合str.translate()来使用
str.replace(‘a’,’b’) 替换，把a替换成b（比如应用到电话号码替换成*）
str.rfind(‘a’) 查找最右边那个匹配值的索引号
str.split(‘’) 将所传入的符号作为分隔符，分割字符串，输出列表
如：
s='www.mengma.com.cn'
print(s.split('.'))
>>>['www','mengma','com','cn']
str.splitlines() 按\n换行来分，跟split(‘\n’)类似，不过这个可能Linux是\n，windows是\r\n来换行，所以有用
str.swapcase() 将大写变小写，小写变大写
str.title() 将字符串的每个字符的首字母大写，变成标题
str.zfill(50) 不够的用zero即0来填充

列表类型

list方法名 功能
list.append() 末尾增加一个元素（其实准备的说应该是是类中的方法）
list.extend() 末尾增加一个列表
list.insert(1,’grape’) 任意位置插入一个元素或列表（是在指定位置元素之前插入，如果指定的位置在列表中不存在，实际上也就是超出指定列表长度，那么这个元素一定会放在列表的最后位置）
list.remove(‘grape’) 删除元素，里面写匹配值
list.pop() 删除一个元素，里面写索引号
list.index() 查找元素的索引号
list.count() 统计元素的个数
list.clear() 清空列表中的所有元素
list.reverse() 反转元素
list.sort() 排序，特殊符号，数字，大写，小写的顺序
list.copy() 复制一份列表（但是是浅拷贝，只拷贝一层，比如列表中的列表就不会被拷贝，浅拷贝的拷贝拷贝的第一层只拷贝了第一层的列表的内存地址）

集合类型

set方法名 功能
set.intersection(list_2) 测试与list_2的交集
set.intersection_update()
set.union(list_2) 测试与list_2的并集
set.difference(list_2) 测试与list_2的差集
set.symmetric_difference(list_2) 测试与list_2的对称差集
set.symmetric_difference_update()
set.issubset(list_2) 是否是list_2的子集
set.issuperset(list_2) 是否是list_2的父集
set.add() 添加单个元素
set.update() 添加多个元素
set.clear()
set.copy()
set.difference_update()
set.discard()
set.isdisjoint() 如果两个集合没有交集就返回为True
set.mro()
set.pop()
set.remove()

文件类型

file方法名 功能
file.read() 读取全部内容
file.readline() 读取一行
file.readlines() 读取内容变成列表
file.write() 写入内容
file.tell() 看看当前指针位置（是以字符为计数的）
file.seek() seek是查找的意思，用于修改指针位置，但是你们不知道你想到的具体位置的字符数，所以你需要先回到0到seek，所以tell和seek是配合在一起使用的
file.encoding() 查看文件编码
file.flush() 强制刷新，将内存的内容写入磁盘（比如在屏幕打印进度条时，一个一个打印输出）
file.truncate() 从所在位置开始截断
