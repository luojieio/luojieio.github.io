发表于 2018-02-28 | 分类于 技术 ， mysql |



用户说明：

  • mysql数据库默认只有一个root用户
  • 用户信息保存在mysql数据库user表中

创建用户

注意：新用户创建后是不能登录的，因为还没有设置权限，所以一般可以使用赋权命令GRANT直接赋权和创建用户一步完成

方式一：通过赋权的方式创建用户

方式二：直接创建用户（仍需后续赋权操作）

说明：

  • 用户名：你将创建的用户名
  • 主机：指定该用户在哪个主机上可以登陆, 如果是本地用户可用localhost, 如果想让该用户可以从任意远程主机登陆,可以使用通配符%
  • 密码：该用户的登陆密码,密码可以为空,如果为空则该用户可以不需要密码登陆服务器

示例

1.创建用户luojie，允许从本机登录，带密码

2.创建用户luojie，允许从192.168.1.101登录，带密码

3.创建用户luojie，允许从任意主机登录，带密码

4.创建用户luojie，允许从任意主机登录，不需要密码

5.创建用户luojie，允许从任意主机登录，不需要密码
